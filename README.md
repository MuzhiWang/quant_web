## Node install

    - Download nvm from https://github.com/coreybutler/nvm-windows/releases 
    - Add system variable
        ` [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Users\Administrator\AppData\Local\nvm", "Machine")`
    - Install LTS
        `nvm install lts`
        `nvm use 22.20.0`
